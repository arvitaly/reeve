Changelog

Release history. Each version ships via Homebrew Cask.

v0.1.10
CalmPopover UI overhaul, settings fix, comprehensive test coverage.
  • Redesigned popover with CalmRow expansion: sparkline panels, inline diagnostics, smart suggestions
  • Fixed settings persistence issue
  • 266 tests across ReeveKit and Reeve targets
v0.1.9
Stability fixes for pressure kill routing and disk overlap.
  • Fixed pressure kill routing to correct target process
  • Resolved disk stat overlap between parent and child processes
  • Added terminateGracefully action kind
v0.1.8
Version display in popover footer.
  • Popover footer now shows current Reeve version
  • Makefile auto-update cask + publish target
v0.1.7
Disk analysis tab, multi-metric menu bar.
  • New Disk tab in main window showing per-app I/O rates
  • Menu bar displays configurable metric (memory, CPU, or disk)
  • DiskScanner: scan-then-act pattern for disk usage analysis
v0.1.6
Fix pressure kill sequencing.
  • Corrected the order of operations when pressure-triggered kill fires
v0.1.5
Pressure tab in main window.
  • New Pressure tab showing kernel-reported memory pressure state
  • Pressure history visualization
v0.1.4
Memory pressure kill policy, terminal tab grouping.
  • Rule engine now supports memory-pressure-triggered kills
  • Terminal.app and iTerm2 tabs grouped under parent process
v0.1.3
Apps tab in main window.
  • Full application list with sorting and search in the main window
  • Process tree hierarchy display
v0.1.2
Fix popup list invisible on some macOS configurations.
  • Resolved rendering issue where popover content was blank on certain display configs
v0.1.1
Fix menu bar icon rendering.
  • Corrected icon sizing and rendering in the macOS menu bar
v0.1.0
Initial public release.
  • Menu bar icon with severity state
  • Popover with application groups, inline actions
  • Floating overlay widget (compact, expanded, pinned, dashboard)
  • Rule engine with auto-kill and auto-renice
  • System pressure bar with kernel-reported memory
  • Hold-to-kill, suspend/resume, lower priority actions
  • Homebrew Cask installation
